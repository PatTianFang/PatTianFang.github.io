在网页中优雅地嵌入 PDF 文件

📅 2026-03-20 demo

如果你经常写作学术论文，或者有一份排版精美的排版笔记，将其转换为网页格式（HTML）往往会丢失原有的优美排版。

因此，最好的方式是：直接在网页中嵌入 PDF 进行展示。

如果您的浏览器不支持直接预览，请 点击此处下载原始 PDF 文件。

如何实现？

只需要在你的 HTML 文章正文中加入以下标签即可：

<embed src="../../pdfs/papers/your-paper.pdf" type="application/pdf" width="100%" height="800px">

此方案极其适合重度 Markdown / LaTeX 用户，你只管专注写作并导出 PDF，剩下的交给 `embed`。
